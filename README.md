# gbc-gyroscope

## Arduino IDE Config

Board: Arduino Mega 2560

Processor: ATMega 2560

Port: Serial 1 (should be automatic)



## Output Example (previous)

Start rotation: 
`1, 2, 3, 4`
Stop

Reverse rotation:
`3, 2, 1, 0, -1, -2, -3`
Stop

Reverse rotation again:
`-2, -1, 0, 1, 2`
